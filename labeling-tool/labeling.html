Underwater Data Labeler

Name:

Place one bounding box of the appropriate color surrounding the entire object. If you can't see all or almost all of an object, don't label it.

Press Submit when you've placed all the boxes on each image.

There are multiple datasets - use the button on the top left to switch between them.

Complete instructions are here: https://ksu-auv-team.github.io/systems/labeling-instructions

Label some data, download the file, and send it to https://goo.gl/forms/i9zQVxzIyPSYcxbm1. Thank you for your contribution!
